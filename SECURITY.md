# Security Policy

## Scope

This is a content-only list of FLOSS projects. It contains no code, no servers, and no credentials. The only meaningful security concern is a link pointing to a malicious or compromised project.

## Reporting a Vulnerability

If you spot a link to a known-malicious project or a project that has been compromised:

1. Open a **GitHub issue**.
2. Include the link and why it should be removed.

## Supported Versions

Only the latest commit on `master` is supported.
